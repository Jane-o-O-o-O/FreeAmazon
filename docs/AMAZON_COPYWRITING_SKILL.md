# Amazon 商品文案生成 Skill

## 目标

为跨境电商商品生成可直接二次编辑的上架文案，重点覆盖 Amazon Listing，但也能迁移到 TikTok Shop、Shopee、Lazada、eBay 和独立站。

## 输出结构

必须输出 JSON 对象，不要输出 Markdown，不要解释过程：

```json
{
  "title": "平台标题",
  "short_title": "短标题",
  "bullet_points": ["卖点1", "卖点2", "卖点3", "卖点4", "卖点5"],
  "description": "详情描述",
  "tags": ["标签1", "标签2"],
  "seo_keywords": ["关键词1", "关键词2"]
}
```

## Amazon Listing 规则

- 标题优先包含核心品类词、关键属性、适用场景和高价值关键词。
- 不编造认证、材质、尺寸、保修、功效、销量、排名或平台背书。
- 不使用夸大词，如 best、guaranteed、No.1、miracle、100% safe，除非输入明确提供。
- 五点卖点要分别覆盖：核心功能、使用场景、材质/结构/规格、用户收益、包装/礼品/售后注意点。
- 描述要自然、可读，适合详情页首屏，不要堆砌关键词。
- 搜索词要偏用户会搜索的词，避免重复品牌词和无意义形容词。
- 如果信息不足，用稳健表达，不要虚构参数。

## 多平台风格

- Amazon：清晰、搜索友好、强调属性和购买理由。
- TikTok Shop：短句、画面感、直播和短视频转化。
- Shopee：性价比、现货、小白易懂、关键词覆盖。
- Lazada：品质、规格、平台搜索词。
- eBay：标题更短，突出品类词、属性词和兼容/适用场景。
- 独立站：品牌感、场景感、信任感和转化导向。

## 语言

按用户指定语言输出。中文要自然，不要机器翻译腔；英文要像真实电商 listing。
